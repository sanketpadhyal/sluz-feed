## 🛠️ Phase 1 – Deep DSA + Core College Subjects  
**(Months 1–6 of Second Year | Sem 3 & Sem 4)**

### 1. DSA – Intermediate → Advanced Level  
**Why medium-heavy?** 60–80% of company coding rounds = medium LeetCode style.

**Timeline & Topic Order** (Follow Striver SDE Sheet + Love Babbar 450 + LeetCode company tags)

**Months 1–2 – Advanced Arrays & Searching Mastery**
- Kadane’s & variants (max circular sum, max product subarray)
- Sliding Window (variable size): longest substring without repeating chars, min window substring
- Two Pointers: 3Sum / 4Sum, container with most water, sort colors / Dutch flag
- Binary Search advanced: rotated sorted array (find pivot + search), kth smallest in sorted matrix
- Prefix Sum + Hashing tricks (subarray sum equals k, longest subarray with sum k)

**Months 3–4 – Linked List, Stack/Queue, Backtracking**
- Linked List: reverse k-group, detect & remove cycle, add two numbers (linked list), copy list with random pointer
- Stack: next greater/smaller (left & right), stock span problem, largest rectangle histogram intro
- Queue: LRU Cache (design), sliding window maximum
- Recursion + Backtracking: subsets II (duplicates), permutations II, combination sum II/III, N-Queens, Sudoku solver, word search II

**Months 5–6 – Trees, BST, Heaps, Graphs Intro**
- Binary Trees: zigzag level order, boundary traversal, max path sum, construct from inorder + preorder/postorder
- BST: validate BST, kth smallest/largest, LCA in BST, delete node in BST
- Heaps: kth largest in stream, merge k sorted lists, top k frequent elements
- Graphs basics: BFS/DFS on grid, number of islands, flood fill, course schedule (topo sort + cycle detection)

**End-of-Phase Targets**
- Total solved: **500–700 problems**
- Breakdown: ~200 easy (revision), **350–450 medium**, 50–100 hard
- LeetCode: 250+ mediums solved, start company-tagged lists (Amazon, Microsoft, Google medium)
- Revision: Re-solve after 7 days, 30 days, 90 days (LeetCode revisit feature or Anki)

**Top Resources (Pick 2–3 max)**
- Striver SDE Sheet + takeUforward YouTube
- Love Babbar DSA Cracker
- Aditya Verma (recursion/backtracking playlists)
- GeeksforGeeks topic practice
- NeetCode 150 / Blind 75 curated lists

**Daily DSA Rules**
- Struggle 20–40 min → hint → more struggle → editorial
- Write clean code + comments + time/space complexity
- Note 1–2 learnings per problem
- Never copy full solution

### 2. Core College Subjects – Interview Depth  
**OOP Deep Dive**
- Inheritance vs Composition (prefer composition)
- Polymorphism (compile-time vs runtime + virtual functions)
- Abstraction vs Encapsulation
- SOLID principles (with real examples)
- Design patterns intro: Singleton, Factory, Builder, Observer

**DBMS Mastery**
- Advanced SQL: RANK/DENSE_RANK/ROW_NUMBER, CTE, recursive CTE
- Joins: self-join, cross, full outer
- Normalization: 1NF–BCNF + anomalies
- Indexing: clustered vs non-clustered, B-tree
- Transactions: ACID, isolation levels, concurrency problems
- Practice: LeetCode Database, HackerRank SQL, StrataScratch

**Operating Systems**
- Process vs Thread
- Scheduling: FCFS, SJF, Priority, Round Robin
- Deadlock: conditions, prevention, Banker’s algorithm
- Memory: paging, segmentation, virtual memory, page replacement (LRU/FIFO)
- Synchronization: mutex, semaphore, producer-consumer

**Computer Networks**
- OSI vs TCP/IP layers
- TCP (3-way handshake, congestion control) vs UDP
- HTTP/HTTPS, DNS resolution, FTP, SMTP
- IP addressing, subnetting, CIDR
- ARP, ICMP basics

**Best Resources**
- Gate Smashers YouTube (Hindi/English – syllabus-aligned)
- GeeksforGeeks articles
- Jenny's Lectures CS/IT
- Selected chapters from Galvin (OS), Forouzan (CN)

**Goal** — Explain any topic 5–10 min + write SQL queries live

### 3. Git & Collaboration Skills
- Rebase vs Merge
- Interactive rebase (squash)
- Stash, cherry-pick, revert/reset
- Pull requests & code review process
- Contribute: Hacktoberfest (Oct), good-first-issue repos
- Branch strategy: feature/xxx, bugfix/xxx

## 🌐 Phase 2 – Full-Stack Web Development Mastery  
**(Months 1–12 – Highest Demand Path in India)**

**Why MERN / Full-Stack?**  
Most fresher jobs & internships in 2026–2027 are web development (MERN most popular, Java Spring for backend roles).

### Frontend Track (Months 1–4)
- HTML5: semantic tags, accessibility, form validation
- CSS3: Flexbox + Grid, animations, transitions, dark mode, mobile-first responsive
- JavaScript ES6+: destructuring, modules, promises → async/await, fetch, error handling
- React.js: JSX, functional components, state/props, hooks (useState, useEffect, useContext, useReducer), lists & keys, conditional rendering
- Routing: React Router v6 (Routes, Route, Link, NavLink, useParams, useNavigate)
- Styling: **Tailwind CSS** (2026 preferred) or Chakra UI / styled-components

### Backend Track (Months 4–8)
- Node.js + Express: routing, middleware, controllers, error handling
- REST APIs: status codes, versioning, pagination, proper structure
- MongoDB: Mongoose schemas, queries, aggregation pipeline basics
- Auth: JWT + refresh tokens, bcrypt, protected routes
- .env, CORS, rate limiting intro

### Full-Stack Integration & Deployment
- Frontend ↔ Backend: Axios/fetch, handling auth tokens
- State management: Context API (preferred for most projects) or Redux Toolkit
- Deployment stack:
  - Frontend → Vercel / Netlify
  - Backend + DB → Render / Railway / Cyclic / Fly.io
  - Full app → Vercel (frontend) + Render (backend)

### Must-Build Intermediate+ Projects (Deploy EVERY ONE!)
1. **Advanced Task / Project Manager**  
   Auth, CRUD tasks, categories, priority, due dates, drag-drop (react-beautiful-dnd), search/filter, dark mode

2. **Full Blog / CMS Platform**  
   User register/login, rich text editor (React-Quill), comments, likes, categories/tags, pagination, admin dashboard

3. **Mini E-commerce Store**  
   Products listing, filters/sorting, cart (DB or local), fake payment flow, orders history, admin product management

4. **Real-time Group Chat App Basics**  
   Socket.io, user login, rooms, online status, message timestamps, typing indicator

5. **Personal Finance / Expense Tracker**  
   Income/expense CRUD, categories, monthly reports, charts (Chart.js/Recharts), CSV export

6. **Dynamic Portfolio + Admin Panel**  
   Showcase projects, contact form (EmailJS), admin login to add/update/delete projects dynamically

**Pro Tip:** Post every deployment on LinkedIn → "Just deployed my full-stack blog app! Live link in comments."

**Top Learning Resources**
- freeCodeCamp Full-Stack Certification
- Harkirat Singh (100xDevs YouTube / videos)
- Roadmap.sh full-stack guide
- The Net Ninja (React + Node series)
- Build in public — share progress!
